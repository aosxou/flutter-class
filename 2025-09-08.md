## 플러터 코드 변경하기 (1.기본 설정)

- Windows PowerShell에 접속
- 'mkdir'명령어 사용하여 디렉토리 생성
- 'cd'명령어 사용하여 생성한 디렉토리로 경로 변경.
- 'flutter create helloworld'를 통해 플러터 데모 홈페이지 생성
- 'code .' 명령어로 VS Code 연동 접속.
- VS Code 접속 후에는 우측 하단에서 연결 프로그램을 Chrome으로 변경

## 플러터 코드 변경하기 (2.홈페이지 타이틀 영문에서 한글로 변경)

- 연동된 접속된 VS Code에서 HELLOWORLD 폴더 내에 있는 'lib'폴더 선택 
- 해당 폴더 안에 'main.dart' 파일 선택
-   home: const MyHomePage(title: 'flutter demo home page') 부분을 찾아서, home: const MyHomePage(title: '플러터 데모 홈페이지')로 변경.
- Windows PowerShell에서 flutter run으로 재실행하면 수정된 내용으로 표시.

## 플러터 코드 변경하기 (3.우측 하단 버튼 한 번 누를 때마다 2씩 증가)

```dart
  void _incrementCounter() {
    setState(() {
      _counter == 1;
    });
  }

  로 되어 있는 코드를
  
  void _incrementCounter() {
    setState(() {
      _counter += 2;
    });
  }  
```
  로 변경하여 저장.
