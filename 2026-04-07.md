## ⚠️ 주요 고려사항

1. **API Rate Limit**
   - OpenRouter 무료 모델의 요청 제한
   - 해결책: 요청 큐잉 및 사용자 안내

2. **재료 인식 정확도**
   - 복잡한 냉장고 환경에서 오류 가능
   - 해결책: 사용자 수동 편집 기능 (Phase 2)

3. **응답 시간**
   - 네트워크 속도 변수
   - 목표: API 응답 10초 이내

4. **개인정보 보호**
   - 사용자 이미지 서버 저장 금지 (즉시 삭제)
   - HTTPS 필수
