# 첫 번째 단계

## 1. pubspec.yaml

Flutter 프로젝트의 환경 설정 파일로,
패키지 정보 / SDK 버전 / dependencies(의존성) / asset 경로 등을 관리.

```dart
name: bamtol_market_app
description: "A new Flutter project."
publish_to: 'none'

version: 1.0.0+1

environment:
  sdk: ^3.9.2

dependencies:
  flutter:
    sdk: flutter
  cupertino_icons: ^1.0.8

dev_dependencies:
  flutter_test:
    sdk: flutter
  flutter_lints: ^5.0.0

flutter:
  uses-material-design: true

  assets:
    - assets/images/yclogo.png
    - assets/svg/
    - assets/svg/icons/

```

## 설명

- name: 프로젝트 이름
- environment: 사용할 Dart 버전
- dependencies: 외부 패키지 (flutter, cupertino_icons 등)
- flutter: 아래의 assets:는 앱에서 사용할 로컬 이미지 / 리소스 등록
- assets/images/yclogo.png → Image.asset으로 불러오는 이미지
- assets/svg/ → svg 전체 폴더 로드
- assets/svg/icons/ → svg 아이콘 폴더 로드

- pubspec.yaml 수정 후 반드시 실행
flutter pub get


## 2. lib/main.dart

Flutter 앱의 진입점(Entry Point) 파일

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        body: Center(
          child: Image.asset("assets/images/yclogo.png"),
        ),
      ),
    );
  }
}
```
## 설명

- main() 함수는 앱 시작 시 가장 먼저 실행되는 함수
- runApp(MyApp()) → MyApp 위젯을 전체 앱으로 렌더링
- MaterialApp → 기본 Material 디자인 구조 제공
- Image.asset(...) → pubspec.yaml에 등록된 로컬 이미지 출력

- 현재는 아주 기본적인 이미지 하나만 보여주는 샘플 구조

## 3. test/widget_test.dart

Flutter 기본 제공 위젯 테스트 파일.
UI가 정상적으로 렌더링되는지 자동 검증하는 용도로 사용.

```dart
import 'package:flutter/material.dart';
import 'package:flutter_test/flutter_test.dart';

import 'package:bamtol_market_app/main.dart';

void main() {
  testWidgets('Counter increments smoke test', (WidgetTester tester) async {
    // Build our app and trigger a frame.
    await tester.pumpWidget(const MyApp());

    // Verify that our counter starts at 0.
    expect(find.text('0'), findsOneWidget);
    expect(find.text('1'), findsNothing);

    // Tap the '+' icon and trigger a frame.
    await tester.tap(find.byIcon(Icons.add));
    await tester.pump();

    // Verify that our counter has incremented.
    expect(find.text('0'), findsNothing);
    expect(find.text('1'), findsOneWidget);
  });
}
```

## 설명

Flutter 기본 템플릿에서 자동 생성된 위젯 테스트 코드

testWidgets() 함수로 UI 동작을 자동 검증

현재 main.dart에는 Counter 기능이 없기 때문에
테스트 실행 시 실패하는 것이 정상

필요 없으면 test 폴더를 삭제해도 앱 실행에는 영향 없음

## 정리


| 파일                        | 설명                           |
| ------------------------- | ---------------------------- |
| **pubspec.yaml**          | 프로젝트 기본 설정 / 패키지 / assets 등록 |
| **lib/main.dart**         | 앱의 진입점, 첫 화면을 만듦             |
| **test/widget_test.dart** | 기본 제공 위젯 테스트 코드(필요하면 삭제 가능)  |

# 두 번째 단계

## 1. lib/main.dart

<역할>
프로젝트의 엔트리 포인트(시작점).
GetX 라우팅을 사용해 첫 번째 화면(App)을 연결.

<주요 기능>
- GetMaterialApp 설정
- 테마(AppBar/Scaffold 컬러) 적용
- 초기 라우트 설정 '/'
- App() 화면으로 이동.

```dart
import 'package:flutter/material.dart';
import 'package:bamtol_market_app/src/app.dart';
import 'package:get/get.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});
  @override
  Widget build(BuildContext context) {
    return GetMaterialApp(
      title: '당근 마켓 클론 코딩',
      initialRoute: '/',
      theme: ThemeData(
        appBarTheme: const AppBarTheme(
          elevation: 0,
          color: Color(0xff212123),
          titleTextStyle: TextStyle(
            color: Colors.white,
          ),
        ),
        scaffoldBackgroundColor: const Color(0xff212123),
      ),
      getPages: [
        GetPage(
          name: '/',
          page: () => const App(),
        )
      ],
    );
  }
}
```

## 2. lib/src/app.dart
<역할>
실제로 화면에 보여지는 첫 번째 UI 화면.
여기서는 프로젝트 로고(yclogo.png)를 가운데에 보여줌.

<주요 기능>
- Scaffold UI 구성
- 이미지 asset 렌더링

```dart
import 'package:flutter/material.dart';

class App extends StatelessWidget {
  const App({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: Center(
        child: Image.asset("assets/images/yclogo.png"),
      ),
    );
  }
}
```

## 3. test/widget_test.dart
<역할>

<주요 기능>
- MyApp 위젯 빌드
- 카운터 증가 테스트 (기본 템플릿)
