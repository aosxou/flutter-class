프리랜서 마케터 포트폴리오 — 작업 세션 정리

작성일: 2026-03-10
목적: 채용 담당자가 30초 안에 핵심 역량을 파악할 수 있는 포트폴리오 웹페이지 기획 및 구조 설계


1. 프로젝트 개요
항목내용제품명프리랜서 마케터 개인 포트폴리오 사이트타겟B2B/B2C 기업 채용 담당자, 스타트업 대표, 에이전시 PM핵심 목표30초 룰 — Above the Fold에서 누구인지 / 무엇을 잘하는지 / 어떤 성과를 냈는지 전달제작 방식단일 HTML 파일 (CSS + Vanilla JS), 외부 프레임워크 미사용배포GitHub Pages / Netlify

2. PRD 핵심 설계 원칙

30초 룰 — 히어로 섹션 하나만 봐도 포지셔닝이 전달되도록 구성
숫자가 말하게 — 추상적 설명 대신 STATS 섹션과 프로젝트 카드에 수치 중심 배치
어디서든 CTA — 히어로, 프로젝트, 컨택 섹션 모두에 액션 버튼 배치


3. 기술 스택
레이어기술마크업HTML5 (시맨틱)스타일CSS3 (Custom Properties, Grid, Flexbox)인터랙션Vanilla JS (ES6+)폰트Google Fonts (Pretendard 또는 Inter)아이콘Lucide Icons 또는 SVG 인라인성과 카운터Intersection Observer API

├── [NAV]          상단 고정 네비게이션

├── [ABOUT]        소개 섹션
├── [STATS]        성과 지표 섹션

├── [CONTACT]      컨택 섹션
└── [FOOTER]       푸터

8. 작업 메모

섹션 ID는 section-{name} 형식, 요소 ID는 {section}-{element} 형식으로 통일
모든 섹션에 aria-labelledby 적용 완료 → 접근성 기반 확보
work-modal-panel은 hidden 상태로 선언 → Phase 3에서 JS 연결 예정
문의 폼 실제 전송은 Formspree 등 서드파티 연동 필요
